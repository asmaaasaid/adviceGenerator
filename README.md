# Advice Generator App

A modern advice generator app built with React and Vite.

## Features

- Generate random advice
- Dark / Light mode
- Save favorite advice
- Copy advice to clipboard
- Responsive design
- React Router navigation
- Loading spinner
- localStorage support

## Built With

- React
- Vite
- React Router DOM
- Context API
- CSS Modules

## Installation

```bash
git clone https://github.com/your-username/advice-generator-app.git

cd advice-generator-app

npm install

npm run dev
```
